<h2>Overview</h2>
This package creates a simple Express boilerplate project for a REST API server
<h2>Installation</h2>
<code>
$ npm install -g create-rest-server
</code>
<h2>Usage</h2>
Create the server:
<br/>
<code>
$ create-rest-server foo
</code>
<br/>
Navigate into the directory
<br/>
<code>
cd foo
</code>
<br/>
Start the server:
<br/>
<code>
node .
</code>
